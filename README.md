Seguidor de linea modelo ABC (versión.II)

Durante el presente año, los integrantes del equipo "ABC" han asumido el compromiso de desarrollar un nuevo diseño del proyecto previo denominado "Robot seguidor de línea ABC I". El rediseño contempla modificaciones estructurales y funcionales orientadas a optimizar el desempeño del sistema, incrementar la precisión en el seguimiento de trayectorias y mejorar la interacción con el usuario.
